# telecom-customer-analytics
Overview:
This project analyzes telecom customer data to identify customer behavior patterns, service utilization trends, revenue insights, and business opportunities.

Business Objective:
The goal is to analyze customer information and generate actionable insights that support customer retention, service optimization, and business growth.

Dataset Information:
The dataset contains:
Customer ID
Customer Type
Region
Service Type
Monthly Charges
Total Revenue
Tenure
Account Status
Payment Method

Project Workflow:
1. Data Collection
Collected customer-related information from telecom databases and consolidated records for analysis.

2. Data Cleaning
Performed:
Duplicate removal
Missing value analysis
Data validation
Standardization

3. Data Analysis
Analyzed:
Customer distribution by region
Revenue by service type
Customer tenure trends
Payment method analysis
Active vs inactive customers

4. Dashboard Creation
Created Excel dashboards displaying:

Total Customers:
Revenue
Customer Segmentation
Regional Analysis
Service Utilization Trends

Key Insights:
Premium service customers generated the highest revenue.
Customers with longer tenure showed higher retention.
Certain regions contributed significantly more revenue than others.
